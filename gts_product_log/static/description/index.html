Product Log

This module will create log. When you Change the fields.

Features

  • This module will help to create a log on change of state of fields
  • It will create log for accounting's journal
  • It will create log for Inventory's operation type
  • It will create log for Inventory's Warehouse
  • It will create log for manufacturing's product bill of material
  • It will create log for Inventory's product
