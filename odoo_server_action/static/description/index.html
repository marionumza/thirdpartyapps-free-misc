Overview

This modules adds menu to create multiple remote servers and option to start, stop and restart. System stores histories of all operations. You have to install the python library paramiko to use this app. If you need any help, please contact abdullah@bisolv.com.

Command for Paramiko Installation : sudo pip3 install paramiko
Activate menu for the user.
