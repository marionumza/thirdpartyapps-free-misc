Backend Multi Barcode

Enhance scanning multi barcode feature of Odoo

Backend Multi Barcode Features:

Normally, Odoo only supports one barcode for one product. But in some case, we need to support multi barcode for one product. Backend Multi Barcode module is really helpful for you in that cases. This module only supports to search a product with multi barcode on BackOffice at line-level.

  • Compliance with Native Odoo
  • Set-up multiple barcode for product at product template or product variants on BackOffice
  • Allowing search product with multi barcode on Sale/Purchase/Inventory...
