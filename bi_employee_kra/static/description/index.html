Human Resource productivity report Employee Performance Evaluation Odoo Apps

HR Employee Performance report KRA/KPA Odoo apps provides the best way to evaluate employee performance regularly, on equal criteria. This module provides two different flows to evaluate the performance of your employee 1.KRA(Key Result Area) 2.Value Rating.

Manage Employee Performance Evolution

One can easily mannage their employee performance evoultion using this module.It provide two ways to evolute Peerformance of employee 1.KRA and 2.Value Rating.

1.KRA(Key Result Area)

Here emplyee can evaluted the KRA after the creation of the KRA from HR.Manager will evaluate that KRA once it is submitted by employee.

KRA Configuration

KRA configuration form contaions predefine questions and their weightage and you can add description too.


Setting KRA on related JOB.

Here you can set the generated KRA on the related JOB,here no need to set it on individule employee.

Create KRA

Here KRA is created for the employees by HR department,have facility to select employee or to just assign to all employee.



Create KRA wizard.

KRA Tree/Form views

Here you can see tree created KRA for all respective employees.


KRA tree view.



KRA form view.

To approv KRA submitted by employee

Once employee submit their respective KRA to manager, submitted KRA will be displayed in "KRA to approve" menu.

Using Value Rating manager will evaluate employee directly.

Value Rating is the another way then KRA to evaluate the employee performance, It got predefined field which can be directly filled by the manager.
