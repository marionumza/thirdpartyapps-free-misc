User Guide

Navigate to Inventory and create a Product from the Products page on the Master Data menu.

Create another Product with the same name.

You will get an error prompt when you click 'Save' showing that this particular Product name must be unique and already exists. (Note that our module trims extra spaces, and matches upper case and lower case initials for the product name before saving it.)
