Quant Package - Package informations in Picking Document

Show informations of the packages in the picking document

Key Highlights

  • Add the dimension and gross weight in Picking Package
  • Show the informations of the packages in the picking document
  • Overview
  • Screenshots

Overview

By installing this module, you will get two extra fields in quant package form to add the dimension and the gross weight of each package of your picking. and also a table of package gonna be shown in the picking document.
