SprintIT Portal Document Access

This app will allow the organizations to give access of portal documents to portal user or not.

Table of contents

FUNCTIONALITY

 

APPLICATION HIGHLIGHTS [back]

Organization can choose to give access to individual partner/customer to show the documents on the portal login under "My Account".
Default the access is not there so, for every customer that checkmark need to check if need to give access to the documents.

 

FUNCTIONALITY [back]


(1) Checkmark is on to see portal documents to logged in user on portal


(1) Under My account logged in user can see his/her documents.
(2) User can see docutments as per access.


(1) Checkmark is off to not allow to see portal documents to logged in user on portal


(1) Under My account logged in user can not see his/her documents.
(2) Warning will be displayed instead.
