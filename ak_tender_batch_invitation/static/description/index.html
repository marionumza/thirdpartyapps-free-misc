Invite Vendors in Batch

Invite multiple vendors to a call for tenders in batch.

Compatible with Odoo Community and Enterprise

Features

Create RFQs in batch within your call for tenders

Instead of creating each RFQ indvidually in your call for tenders, add the list of vendors you would like to invite and the app will create the RFQs for you in batch.

Send RFQs to invited vendors in email

During the invitation, the app will provide the option to send out RFQs using the default purchase RFQ email template, with the RFQ document attached.
