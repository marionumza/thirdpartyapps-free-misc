Chatter Attachment Comment

With help of chatter attachment comment Odoo apps, you can add comment on chatter attachment section on required attachment file. It is very helpful you to identify requirement of that attachment on chatter section by adding short comment.

Watch on YouTube

Chatter Attachment Comment Steps

Step 1:- Click on chatter attachment main button

Step 2:- Click on chatter attachment add button and attach attachment

Step 3:- Click on chatter attachment comment button. It will open comment dialog box.

Step 4:- Add comment on description field on comment dialog box and click on save button

Step 5:- Once comment saved on system. It will show below attachment name on attachment box.
