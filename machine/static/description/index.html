Plan, log and inform about machine changes

Can log planned/done changes for machines to always know what was done. Can use groups to group machines having same purpose (e.g. testing machines, production machines etc.). Using same groups, can inform related partners about planned changes, where mail is automatically generated according to what is planed to be done and then sent to needed recipients.

Integrated with Odoo messaging

Information messages about scheduled changes are saved on respective machine messaging chatter to for easier communication.
